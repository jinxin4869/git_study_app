# Git Learning App - Walkthrough

## 概要
Gitの基本操作から応用までを学べるWebアプリケーションを作成しました。
Next.js + React + Tailwind CSS + Framer Motion で構築されており、実際のGitコマンドをシミュレートしながら、視覚的に理解を深めることができます。

## 機能一覧

### 1. ターミナルシミュレータ
- `git init`, `add`, `commit`, `branch`, `checkout`, `status`, `log` などの基本コマンドをサポート。
- リアルタイムでコマンドの結果を表示。

### 2. ビジュアライザ
- コミットグラフをリアルタイムで描画。
- ブランチ、HEADの位置を視覚的に確認可能。
- アニメーションによる直感的な理解。

### 3. シナリオモード
- **Level 1 (初級)**: リポジトリ作成からコミットまで。
- **Level 2 (中級)**: ブランチ作成と切り替え。
- **Level 3 (上級準備)**: マージとコンフリクト（現在はFast-forwardマージの準備段階）。
- **Level 4 (実務)**: 作業の一時退避 (`git stash`)。
- **Level 5 (実務)**: 変更の取り消し (`git reset --soft`)。
- **Level 6 (実務)**: 過去の調査 (Detached HEAD)。

## 使い方
1.  左側のサイドバーからレベルを選択します。
2.  中央のターミナルにコマンドを入力します（例: `git init`）。
3.  右側のビジュアライザで状態の変化を確認します。
4.  ゴールを達成すると、メッセージが表示されます。
